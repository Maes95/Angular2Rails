# Angular2Rails
Angular 2 + Rails Start Proyect

## Ruby on Rails SetUp

Make sure you have all Ruby gems dependencies installed:

    bundle install

Create and populate the dev db:

    rake db:setup

Run rails server:

    rails s

## Angular 2 SetUp

Install dependecies:

    cd Frontend
    npm install

Run Angular 2 client with a proxy

    ng serve --proxy http://localhost:3000
